LangChain Expression Language (LCEL)
=====================================
LCEL is a declarative way to compose chains using the pipe operator (|).
It is the recommended way to build chains in LangChain.

Basic syntax:
    chain = prompt | llm | output_parser

Each component must implement the Runnable interface, which provides:
    invoke(input)       - synchronous single call
    stream(input)       - stream output tokens
    batch(inputs)       - process multiple inputs in parallel
    ainvoke(input)      - async version of invoke
    astream(input)      - async streaming
    abatch(inputs)      - async batch processing

Example - simple RAG chain:
    from langchain_core.prompts import ChatPromptTemplate
    from langchain_core.output_parsers import StrOutputParser
    from langchain_openai import ChatOpenAI

    prompt = ChatPromptTemplate.from_template(
        "Answer the question based only on the context:\n{context}\nQuestion: {question}"
    )
    llm = ChatOpenAI(model="gpt-4o-mini")
    chain = prompt | llm | StrOutputParser()
    result = chain.invoke({
        "context": "Paris is the capital of France.",
        "question": "What is the capital of France?"
    })

LCEL supports parallel execution with RunnableParallel:
    from langchain_core.runnables import RunnableParallel, RunnablePassthrough

    parallel = RunnableParallel(
        answer=chain,
        question=RunnablePassthrough()
    )

RunnableLambda wraps any Python function as a Runnable:
    from langchain_core.runnables import RunnableLambda

    def format_input(x):
        return x.upper()

    chain = RunnableLambda(format_input) | llm | StrOutputParser()

Binding runtime arguments with .bind():
    # Bind stop sequences to an LLM
    llm_with_stop = llm.bind(stop=["\nObservation"])

Configurable chains allow switching components at runtime:
    from langchain_core.runnables import ConfigurableField

    llm = ChatOpenAI(model="gpt-4o-mini").configurable_alternatives(
        ConfigurableField(id="llm"),
        default_key="openai",
        groq=ChatGroq(model="llama-3.1-8b-instant"),
    )

Streaming example:
    for chunk in chain.stream({"context": ctx, "question": q}):
        print(chunk, end="", flush=True)
